Dodatnie liczby całkowite można podzielić na sumy różnych sześcianów dodatnich liczb całkowitych, czasami na wiele różnych sposobów. Państwa funkcja powinna znaleźć i zwrócić malejącą {% if is_js %}tablicę{% else %}listę{% endif %} różnych sześcianów, których suma jest równa podanej dodatniej liczbie całkowitej n. Jeśli nie można wyrazić n jako sumy różnych sześcianów, proszę zwrócić {% if is_js %}null{% else %}None{% endif %}.

Jeśli n pozwala na kilka podziałów na sumy różnych sześcianów, funkcja musi zwrócić leksykograficznie najwyższe rozwiązanie, które zaczyna się od największej możliwej pierwszej liczby, a następnie leksykograficznie najwyższą reprezentację reszty liczby n-a*a*a. Na przykład, wywołana z n = 1729, funkcja ta powinna zwrócić [12, 1] zamiast [10, 9].
